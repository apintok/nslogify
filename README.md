# NSLogify

[![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/ISC)

-   :computer: Small Extension to **Pretty Print** NetSuite Script Execution Logs.
-   :earth_africa: **Available for Firefox!**. I personally use Mozilla Firefox to work with NetSuite and I don't like the way the SuiteScript logs are presented. So I decided to create this Extension. Soon to be available on the [addons page](https://addons.mozilla.org/en-GB/firefox/collections/4757633/webdeveloper/).
-   :pushpin: **Initial Version Goals** Are to format SuiteScript Logs that are either Arrays or Objects.
-   :construction: **This project is still under development!** :construction: Working still to have it published for any one who wants to use it. There are a lot of things to consider and I am trying to make sure the first version at least works.
-   :alien: **Open** for contributions. Just let me know if you want to.
